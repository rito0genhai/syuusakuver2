＊＊＊初めに＊＊＊

　ロックマンXDiVEの二次創作ゲームです。
　ただし、ストーリーはほぼありません。
　１時間くらいで終わると思います。


　＊＊＊操作方法＊＊＊

　　上下：選択
　　Z：決定
　　X：キャンセル
　　ぽちぽちしていくだけで進みます。ただし戻れず進むだけ。前進あるのみ。


　＊＊＊注意＊＊＊
　
　このゲームで起きたあらゆる事柄において、製作者は一切責任を負いません。
　実況や動画配信などは禁止です。


　＊＊＊他＊＊＊

　・メンバー入れ替え：必ず３人パーティ、ただしゲイトは外せません。
　　　　　　　　　　　レベルアップがぎりぎりなので最初から固定にした方がいいかも。
　　　　　　　　　　　最初のゲイト、エックス、ゼロが一番バランスいいと思います。
　　　　　　　　　　　ゲイト、ゼロ、アイコがわりといばらの道。
　・装備・スキル：装備はランダムエンチャントです。
　　　　　　　　　固有スキル一つ、装備の【メモリ】によって２つまで追加できます。
　　　　　　　　　おすすめスキルはエックスかダイナモに【かばう】です。
　・バランス：上に書いてますが、【かばう】があるかどうかで結構変わります。
　　　　　　　テストプレイしてたらもうよくわかんなくなってきた。助けて






　＊＊＊後記＊＊＊

　色々シンプルすぎるし短いしですが、
　　「うひょーノンフィールド作ったぜー！」
　とめちゃくちゃ嬉しいです。コモンを公開してくださった方々に足向けられない！
　うんうん唸りながらでも作れた！やった！
　前回の初制作では
　　・メンバーは最小限。絵も既存のやつで新しく描かない。
　　・ゲームバランスとかよくわからないからとりあえずで。
　　・UIとかシステムとか演出も凝ろうとせずにデフォルトor最小限で。
　　・とにかく完成させる。これが最優先！
　を念頭に置いてましたが今回それ＋
　　・シンプル簡単でもノンフィールドの体裁で
　　・メンバーを交代できるように。
　を意識してました。結果的にちょっっっとだけツールのことわかった気が…する。２％くらい。
　バグとか意見とか何かあれば一言いただけると嬉しいです。
　でも実況はやめてね。あと改造とか他所への公開とか。普通に遊んでほしいです。

　今ちびちび作ってるの、いつ完成するのやら…。こっちも習作のはずが…。
　でもでも、夢はでっかくゲくんが活躍してるRPG。
