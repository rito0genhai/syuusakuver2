＊＊＊初めに＊＊＊

　ロックマンXDiVEの二次創作ゲームです。
　ただし、ストーリーはほぼありません。
　１～２時間くらいで終わると思います。


　＊＊＊操作方法＊＊＊

　　上下：選択
　　Z：決定
　　X：キャンセル
　　ぽちぽちしていくだけで進みます。ただし戻れず進むだけ。前進あるのみ。


　＊＊＊ゲーム進行＊＊＊

　　進行：バトルしたり装備手に入れたり能力アップしたりします。
　　道具・装備：そのまま。
　　換金：アイテム売却。購入はありません。
　　研究：敵からドロップした素材でメモリ制作します。
　　強化：ゼニーを消費して経験値を得ます。
　　記録：まんまセーブ。ファイルは一つです。
　　設定：これもそのまんま。


　＊＊＊注意＊＊＊
　
　このゲームで起きたあらゆる事柄において、製作者は一切責任を負いません。
　実況や動画配信などは禁止です。


　＊＊＊他＊＊＊

　・キャラについて：必ず３人パーティ、ただしゲイトは外せません。各自、一つ固有特技を持ってます。

　　　　　　　　　　ゲイト：特殊攻撃特化（魔法みたいなの）。防御は紙。
　　　　　　　　　　エックス：防御高め、特殊防御上げたら壁にも可
　　　　　　　　　　ゼロ：攻撃特化。防御は紙。
　　　　　　　　　　アクセル：スピード特化攻撃。スキルは敵の攻撃無効。
　　　　　　　　　　エイリア：特攻と特防。敵の防御ちょっと下げるデバフ持ち。
　　　　　　　　　　ダイナモ：防御ちょっと高いアタッカー。多分一番使いやすいです。
　　　　　　　　　　アイリス：装備に恵まれないとキツいカウンター。防御はきもち高め。
　　　　　　　　　　渾然：壁。クリアしやすくなるけど攻撃には期待できない。
　　　　　　　　　　マーティ：スピード特化特攻。壁にバフかけたら鉄壁になります。
　　　　　　　　　　アイコ：特攻アタッカー。特防もわりと高め。

　　　　　　　　　　ゲイト、渾然、他アタッカーが一番クリアしやすいと思います。

　・装備・スキル：装備はランダムエンチャントです。
　　　　　　　　　スキルは固有スキル一つ、装備の【メモリ】によって２つまで追加できます。

　・バランス：【かばう】があるかどうかで結構変わります。
　　　　　　　強化と研究を行ってたらどのメンバーでもクリアできると思います。
　　　　　　　逆に難易度ぬるすぎ！て方は上の要素省いてみていただけると…。
　　　　　　　テストプレイしてたらもうよくわかんなくなってきた。助けて
　・既知の不具合：強化でレベルアップした時の文字が切れる現象。直せない…。






　＊＊＊後記＊＊＊

　色々シンプルすぎるし短いしですが、
　　「うひょーノンフィールド作ったぜー！」
　とめちゃくちゃ嬉しいです。コモンを公開してくださった方々に足向けられない！
　うんうん唸りながらでも作れた！やった！
　前回の初制作では
　　・メンバーは最小限。絵も既存のやつで新しく描かない。
　　・ゲームバランスとかよくわからないからとりあえずで。
　　・UIとかシステムとか演出も凝ろうとせずにデフォルトor最小限で。
　　・とにかく完成させる。これが最優先！
　を念頭に置いてましたが今回それ＋
　　・シンプル簡単でもノンフィールドの体裁で
　　・メンバーを交代できるように。
　を意識してました。結果的にちょっっっとだけツールのことわかった気が…する。２％くらい。
　バグとか意見とか何かあれば一言いただけると嬉しいです。
　でも実況はやめてくださいね。あと改造とか他所への公開とか。普通に遊んでほしいです。

　今ちびちび作ってるの、いつ完成するのやら…。こっちも習作のはずが…。
　でもでも、夢はでっかくゲくんが活躍してるRPG。


　（追記）……で、メンバー入れ替えが微妙なので入れ替え無くしてその分メンバー増やしました。
　それに合わせて元々薄いカルピスより薄かったシナリオがほぼ水状態に。
　でも、編成したキャラによって場面で３行くらいのイベント挟みます。絵も新しく描き直してみました。
　階層も増やして３００に。クリアまで１時間ちょいくらいにはなりました。
　楽しんでいただけたら嬉しいな…。


　（さらに追記）選択肢が１０個までいけるということで、更にメンバー増やしました。
　エイリアとヴィアはすんなり決まったけど、マリノかマーティか悩み、マリノやとアクセルともろ被るなてことでマーティに。
　階層は最初800、もしくは999て思ったけどブラウザ公開やし～てことで400に。
　ランダムレベルアップを強化項目増やした代わりにステアップにしたり。
　結構変わったんじゃないかな～と思います。
　ちなみに、ゼニーは一定額集まると特殊エンド（セリフ変わるだけ）になったりします。
